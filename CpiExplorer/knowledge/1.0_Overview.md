# CpiExplorer — Overview

CpiExplorer is a browser-based tool for visualizing and analyzing U.S. Consumer Price Index (CPI) data. It supports:

- **Monthly [CPI-U NSA](../../knowledge/DATA_DICTIONARY.md#cpi-nsa)** — full history from January 1913, sourced from [E4: BLS Public API](../../knowledge/DATA_DICTIONARY.md#e4) and stored in [S8: CPI_history.csv](../../knowledge/DataStores.md#s8).
- **Monthly [CPI-U SA](../../knowledge/DATA_DICTIONARY.md#cpi-sa)** — same source, seasonally adjusted series.
- **Daily [Ref CPI](../../knowledge/DATA_DICTIONARY.md#ref-cpi)** — available from April 1998 (TIPS inception), sourced from [S3: RefCPI.csv](../../knowledge/DataStores.md#s3).
- **Multi-Source Comparison** — select any combination of data sources to overlay on a single chart.
- **Logarithmic Scale** — toggle between linear and logarithmic vertical axes for better visualization of long-term trends.

---

## 1.1 Data Flow (Level 1 DFD)

```
E4 (BLS API) ──► fetchCpiHistory.js ──► S8: bls/CPI_history.csv ──┐
                                                                    ├──► CpiExplorer App ──► User
E2 (TreasuryDirect) ──► fetchRefCpi.js ──► S3: RefCPI.csv ─────────┘
```

**Ingestion** (server-side scripts, run monthly):
- `scripts/fetchCpiHistory.js` → [S8](../../knowledge/DataStores.md#s8)
- `scripts/fetchRefCpi.js` → [S3](../../knowledge/DataStores.md#s3)

**Presentation** (browser, no server):
- `CpiExplorer/index.html` fetches S8 and S3 from Cloudflare R2 at page load.

---

## 1.2 Core Processes

### P1 — Load Data
On page load, fetch and parse [S8](../../knowledge/DataStores.md#s8) (CPI history) and [S3](../../knowledge/DataStores.md#s3) (Ref CPI) from R2. Parse into typed in-memory arrays. Display status (loading / last data date / error).

### P2 — Apply Controls
React to user control changes (data source, display mode, date range, rolling window). Re-run the selected calculation from `calc.js` on the in-memory data. No re-fetch required.

### P3 — Render Chart
Pass the calculated series to `chart.js`, which renders or updates a Chart.js time-series line chart. Support zoom/pan via mouse wheel and keyboard (arrow keys, +/−) using the shared `chart-keys.js` utilities.

### P4 — Stats Strip
After each render, compute and display summary statistics for the currently visible date range: current value, period change, annualized [CAGR](../../knowledge/DATA_DICTIONARY.md#cpi-cagr), peak, and trough.

## 1.3 File Structure

```
CpiExplorer/
├── index.html          ← Shell: header, sidebar controls, chart canvas, stats strip
├── src/
│   ├── data.js         ← P1: fetch + parse S8 and S3; normalize to common row shape
│   ├── calc.js         ← P2: all CPI transformations (Index, YoY, MoM, Rolling, P2P, CAGR)
│   ├── chart.js        ← P3: Chart.js lifecycle (create, update, zoom/pan reset)
│   └── app.js          ← Orchestration: wires controls → calc → chart; manages state
└── knowledge/
    ├── 1.0_Overview.md     ← this file
    └── 2.0_Technical_Spec.md
```
